# Deloitte Data Analytics Project

## Overview
This project is based on a Deloitte Data Analytics simulation. It includes analysis of machine downtime and gender pay equality.

## Tools Used
- Tableau
- Excel

## Tasks
- Built a dashboard to analyze factory downtime
- Identified machine failure patterns
- Classified gender pay equality into categories (Fair, Unfair, Highly Discriminative)
